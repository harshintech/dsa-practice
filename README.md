# LeetCode Playground 🧠🚀

A personal playground for practicing **LeetCode problems** with clear explanations and optimized solutions.

This repository focuses on **understanding approaches**, writing **clean code**, and improving **problem-solving skills**.

---

## 📌 What you'll find here

- ✅ LeetCode problems with **step-by-step explanations**
- 🧩 Focus on common **DSA patterns**
- ✍️ Markdown-based solutions (approach + code in one place)
- ⚡ Optimized solutions with time & space complexity

---

## 🛠 Tech Stack

- **Language:** Java ☕
- **Format:** Markdown (`.md`)
- **Platform:** LeetCode
